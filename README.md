# E-Commerce (React + Vite)

## Project location

This app is inside the nested folder:

`E_Commerce/E-Commerce`

If you run npm commands from `E_Commerce` (one level above), npm will fail with "package.json not found".

## Required dependencies

Runtime dependencies:

- react
- react-dom
- react-router-dom
- react-icons

Dev dependencies:

- vite
- @vitejs/plugin-react
- eslint
- @eslint/js
- eslint-plugin-react-hooks
- eslint-plugin-react-refresh
- globals

## First-time setup (after copy/paste)

### Option 1: Enter project folder first

```bash
cd E-Commerce
npm install
npm run dev -- --host
```

### Option 2: Run from parent folder

```bash
npm --prefix "c:/Users/lakha/OneDrive/Desktop/E_Commerce/E-Commerce" install
npm --prefix "c:/Users/lakha/OneDrive/Desktop/E_Commerce/E-Commerce" run dev -- --host
```
